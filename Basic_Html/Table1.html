1. HTML

Primary Use case Key Features/Benifits
Building the structure of Web Pages
  • Simple and easy to learn
  • Compitable with all web browsers
  • Allows for semantic markup

2. CSS

Primary use case Key Features/Benifits
Styling the web page
  • Animation and Effects: Engagement, Interactivity, Dynamic, Visual appeal, Enhancement.
  • Efficiency: Optimization, Performance, Streamlining, Conciseness, Resourcefulness
  • CSS provides a wide range of benefits that contribute to a better user experience
  • SEO (Search Engine Optimization): CSS can improve your website's visibility in search engines by making it easier for search engine bots to crawl and index your content.

3. Javascript

Primary use case Key features/Benifits
Adding interactivity to web pages
  • Simple and easy to learn
  • Compatible with all web browsers
  • Allows for dynamic and responsive web pages
  • Provides access to the Document Object Model (DOM) for manipulating web page elements
