Features of HTML5
  • Simple and clean code = HTML5 provides new syntactic features and helps in writing simple code
  • Multimedia support = without using any plugin HTML5 supports video and audio playback.
  • Offline support = HTML5 allows web browsers to store data locally on the users device enables the user to work offline.
  • Cross-Platform-Compatibility = HTML5 is designed to work on various devices including desktops, laptops, tablets, smartphones etc.
  • Improved semantics = HTML5 introduces new elements like < header >, < footer >, < main >, < nav > etc to better define structure of web pages, helps in search engine optimization.
