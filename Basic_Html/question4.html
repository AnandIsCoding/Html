1. h1 = heading 1 , defines the most important heading
  2. h1 to h6 = heading in decreasing boldnes or weight
  3. p = paragraph tag defines a paragraph of text
  4. a = anchor tag defines a hyperlink
  5. ul = unordered list, items of unordered list is arranged in unordered fashion (bullet points)
  6. ol= ordered list, items of ol is arranged in order 1,2,3 or a,b,c etc
  7. form = Defines an HTML form for user input.
  8. button = Defines a clickable button.
